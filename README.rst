=========
propulate
=========


Parallel propagator of populations.


Description
===========

Evolutionary hyperparameter-optimization in MPI-parallelized fashion.
For usage example see scripts.

Installation
============

Pull and run '''pip install -e .'''' or '''python setup.py develop'''
